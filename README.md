# SisCon

## Descrição
SisCon é um sistema de controle para impressão 3D comercial, desenvolvido em .NET 9, que visa facilitar o gerenciamento de impressões 3D, permitindo o cadastro de clientes, produtos e ordens de serviço, além dos custos de materiais, insumos e energia elétrica.
